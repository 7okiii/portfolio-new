Dynamic Dashboard

HTML, Sass, Bootstrap Javascript, jQuery, PHP, MySQL, GitHub

A dynamic dashboard for the educational institution. PHP is used to route pages, hash and salt techniques are used for secure data storing. Allows school administrators to register new students and teachers, allows for the publishing of new courses, provides the ability to appoint teachers for each course, and makes it possible to appoint new admin users. For class size, one can set maximum and minimum capacities. It also allows teachers to mark and leave comments to aid students in their learning and progress. Students can see these marks and comments from teachers. Also, students can enroll in a course and check the price and details for each course.

Online Shopping System

HTML, Sass, Angular, Javascript, jQuery

AngularJS’s logic validates each input box by using ng-model access to the element and displays a modal box for each item when the item is clicked. jQuery is used to make a fade-out animation when items are added to the shopping cart. Customers can choose the number of items and add them to a shopping cart. Furthermore, customers can check the details and total price, edit the number of items, and delete items from the shopping cart.

Responsive Website

HTML, Sass, Bootstrap, Javascript, jQuery

A fully responsive landing page for musicians. It uses Bootstrap and the media query method, as well as Javascript to validate each input box by regular expression. Also, it displays different contents depending on what button is clicked. The viewer can see information about the musician, watch a music video by a provided link and check the event and concert date. Viewers can contact musicians through the validated contact form.
